# My House Roofing and Construction Works

Modern digital presence for My House Roofing and Construction Works, built with Next.js and Tailwind CSS.

## Features
- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **Modern UI**: Clean, trustworthy aesthetic with high-conversion elements.
- **Performance**: Built on Next.js for speed and SEO optimization.

## Tech Stack
- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (Icons)

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the browser:**
   Navigate to [http://localhost:3000](http://localhost:3000).# mhroofingandconstructionworks
